# pinewall.github.io

技术文档站点，基于 GitHub Pages 和 Jekyll 构建。

## 内容

- [OpenClaw 基本原理](openclaw.md) - 开源机械爪的结构和控制原理
- [Nanobot 纳米机器人基本原理](nanobot.md) - 纳米尺度机器人的物理原理和应用
- [LLM 大语言模型基本原理](llm.md) - Transformer 架构和大语言模型训练生成原理

## 本地运行

```bash
bundle install
bundle exec jekyll serve
```

然后访问 http://localhost:4000
